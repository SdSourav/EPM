# EPM Exam Prep

This repo now serves only the study page at `/study/`.

The page is fully static and lives at `study/index.html`, so it can be opened
directly or hosted on free static hosting.

## Run locally

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000/study/
```

Or open the file directly in a browser:

```text
study/index.html
```

## Free deployment

For GitHub Pages, publish this repository and open:

```text
https://<your-username>.github.io/<repo-name>/study/
```
